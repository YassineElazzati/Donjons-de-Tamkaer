Guide de survie Java

De la syntaxe Python que vous maîtrisez à la syntaxe Java que vous allez apprendre — toutes les équivalences essentielles.

✦ ❖ ✦
Vous connaissez déjà tous ces concepts en Python. En Java, la logique est identique, mais la syntaxe est plus stricte : tout doit être typé, chaque fichier contient une classe, et la visibilité (public, private) est un vrai mot-clé du langage. Voici les équivalents directs.

Une classe simple

Python
class Produit:
    def __init__(self, nom, prix):
        self.nom = nom
        self.prix = prix

    def afficher(self):
        print(f"{self.nom} : {self.prix}€")
Java
public class Produit {
    private String nom;
    private double prix;

    // Constructeur (remplace __init__)
    public Produit(String nom, double prix) {
        this.nom = nom;
        this.prix = prix;
    }

    public void afficher() {
        System.out.println(nom + " : " + prix + "€");
    }
}

Ce qui change par rapport à Python

  • self devient this (et est souvent optionnel)
  • __init__ devient un constructeur qui porte le nom exact de la classe
  • Chaque attribut doit être déclaré avec son type en haut de la classe : private String nom;
  • private est un vrai mot-clé (pas une convention avec _)
  • Chaque instruction se termine par un point-virgule ;
  • Les accolades { } remplacent l'indentation
  • print() devient System.out.println()

Une méthode qui retourne une valeur

Python
def calculer_ttc(self):
    return self.prix * 1.20
Java
public double calculerTTC() {
    return prix * 1.20;
}
  • En Java il faut indiquer le type de retour avant le nom : double, String, boolean, int…
  • Si la méthode ne retourne rien, utilisez le type spécial void
  • Convention de nommage : camelCase (calculerTTC) et non snake_case

Les getters et setters

Java n'a pas de @property. On écrit explicitement des méthodes getNom() et setNom(...). C'est plus verbeux mais c'est la norme absolue dans tout code Java professionnel.
Python (avec @property)
@property
def prix(self):
    return self._prix

@prix.setter
def prix(self, valeur):
    if valeur < 0:
        raise ValueError("Prix invalide")
    self._prix = valeur
Java
public double getPrix() {
    return prix;
}

public void setPrix(double valeur) {
    if (valeur < 0) {
        throw new IllegalArgumentException("Prix invalide");
    }
    this.prix = valeur;
}

L'héritage

Python
class Ennemi(Personnage):
    def __init__(self, nom, pv, xp):
        super().__init__(nom, pv)
        self.xp = xp
Java
public class Ennemi extends Personnage {
    private int xp;

    public Ennemi(String nom, int pv, int xp) {
        super(nom, pv);
        this.xp = xp;
    }
}
  • (Parent) devient extends Parent
  • super().__init__(...) devient simplement super(...) et doit être la première ligne du constructeur

Classes abstraites et méthodes abstraites

Python
from abc import ABC, abstractmethod

class Personnage(ABC):
    @abstractmethod
    def attaquer(self):
        pass
Java
public abstract class Personnage {
    public abstract void attaquer();
}

Les interfaces (pattern que Python n'a pas vraiment)

Une interface en Java, c'est un contrat : une liste de méthodes qu'une classe s'engage à implémenter. Très utile pour votre interface Objet (arme, armure, potion) ou pour le pattern Observer.
Java — déclarer une interface
public interface Objet {
    String getNom();
    int getValeur();
    void utiliser(Heros heros);
}
Java — implémenter une interface
public class Potion implements Objet {
    private String nom;
    private int valeur;

    @Override
    public String getNom() { return nom; }

    @Override
    public int getValeur() { return valeur; }

    @Override
    public void utiliser(Heros heros) {
        heros.soigner(valeur);
    }
}

Créer un objet et appeler ses méthodes

Python
clavier = Produit("Clavier", 49.99)
clavier.afficher()
print(clavier.prix)
Java
Produit clavier = new Produit("Clavier", 49.99);
clavier.afficher();
System.out.println(clavier.getPrix());
  • Le mot-clé new est obligatoire pour créer un objet
  • On déclare le type de la variable à gauche : Produit clavier = ...
  • Pour lire un attribut privé, on passe par son getter : clavier.getPrix()

Les collections (équivalent des listes Python)

Python
ennemis = []
ennemis.append(Gobelin())
for e in ennemis:
    e.attaquer()
Java
List<Ennemi> ennemis = new ArrayList<>();
ennemis.add(new Gobelin());
for (Ennemi e : ennemis) {
    e.attaquer();
}
  • List<Ennemi> signifie "une liste d'Ennemi" : le type entre < > est obligatoire
  • Il faut importer la classe en haut du fichier : import java.util.ArrayList; et import java.util.List;
  • .append() devient .add()

Le point d'entrée du programme

Python
if __name__ == "__main__":
    main()
Java
public class Main {
    public static void main(String[] args) {
        // point d'entrée du programme
        Jeu jeu = new Jeu();
        jeu.lancer();
    }
}

Les types primitifs à connaître

  • int — nombre entier (ex : PV, niveau, or)
  • double — nombre à virgule (ex : prix, probabilités)
  • boolean — true ou false (minuscules, contrairement à Python)
  • String — chaîne de caractères (attention, la majuscule est obligatoire)
  • char — un seul caractère, entre apostrophes : 'A'
